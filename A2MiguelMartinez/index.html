Conceptos Básicos de Git:

Git es un sistema de control de versiones distribuido que te permite gestionar y controlar cambios en tu código. Algunos conceptos esenciales incluyen:

  • Repositorio
  • Commit
  • Ramificación (Branching)
  • Merge (Fusión)
  • Conflicto de Fusión
  • Rebase
